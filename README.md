# content-insight-scraper

A Python tool to scrape and analyze content insights from websites. Designed for researchers, marketers, and data enthusiasts who want quick visibility into online content trends.

## Current Status
- Version: Alpha
- Functionality: Basic scraping from one website
- Actively under development

## Features
- Scrapes headlines, meta descriptions, and article content
- Output structured PDF

## Installation
1. Clone the repo:
```
git clone https://github.com/tothlevivok/content-insight-scraper.git
```

2. Install dependencies
```bash
pip install -r requirements.txt
  ```
3. Run:
```bash
python scraper.py
```
